# CreativeGrowth

## プラグイン説明

https://github.com/user-attachments/assets/b24d1bdb-9aa3-4360-a098-9455c4371592



クリエイティブでの木の設置が楽になる

## 使い方

- クリエイティブ状態で、シフトしながら 骨粉をクリック
- 木の種類をクリックすることで プレイヤーが立っている場所に、木を生成

## 開発環境

- Minecraft Version : 1.20.1
- Kotlin Version : 1.8.0
